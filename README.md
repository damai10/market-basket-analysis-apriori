# 🛒 Market Basket Analysis using Apriori Algorithm

## 📌 Project Overview
This project analyzes retail transaction data to identify frequently purchased product combinations using the Apriori algorithm.

## 🎯 Objective
To uncover product associations and generate insights for cross-selling and recommendation strategies.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Mlxtend
- Jupyter Notebook

## 🔍 Approach
1. Data Cleaning
2. Basket Transformation
3. Apriori Algorithm
4. Association Rule Mining

## 💡 Key Insights
Identified strong product associations that can be used to improve sales strategies and customer experience.
